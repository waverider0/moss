\[ f''(x) = 3x - 1 \]

\[ f'(2) = 4 \]

\[ f(2) = 2 \]

\[ \text{Find} [f(x)]^2 \]

Answer

\[ [f(x)]^2 = \left(\frac{1}{2}x^3 - \frac{1}{2}x^2\right)^2 \]

Solution

\[ f'(x) = \int(3x - 1)\,dx = \frac{3}{2}x^2 - x + C_1 \]

\[ f'(2) = 4 \implies C_1 = 0 \]

\[ f(x) = \int\left(\frac{3}{2}x^2 - x\right)\,dx = \frac{1}{2}x^3 - \frac{1}{2}x^2 + C_2 \]

\[ f(2) = 2 \implies C_2 = 0 \]

\[ \boxed{[f(x)]^2 = \left(\frac{1}{2}x^3 - \frac{1}{2}x^2\right)^2} \]
